# Politica de Privacidad - Indalo Padel

**Fecha de entrada en vigor:** 10 de abril de 2026  
**Ultima actualizacion:** 10 de abril de 2026

Indalo Padel (la "App") es una aplicacion para reservar pistas y organizar partidos de padel.

Esta Politica de Privacidad explica que datos tratamos, para que los usamos, con quien se comparten y que derechos tienes como usuario.

## 1. Responsable del tratamiento

- **Responsable:** Indalo Padel
- **Email de contacto para privacidad:** `sergio.hlara84@gmail.com`

## 2. Datos que recopilamos

Tratamos los siguientes tipos de datos:

1. **Datos de cuenta**
- Nombre
- Correo electronico
- Contrasena (almacenada en formato hash, no en texto plano)

2. **Datos de perfil de jugador**
- Nombre visible (`display_name`)
- Nivel de juego y preferencias (lado de juego, disponibilidad)
- Biografia y avatar (si los aportas)

3. **Datos de uso funcional de la App**
- Reservas (fecha, hora, pista, sede, estado, notas)
- Partidos (creacion, uniones/salidas, estado, descripcion)
- Favoritos y valoraciones/comentarios entre jugadores

4. **Datos tecnicos minimos**
- Registros tecnicos del servidor (logs de peticiones y errores)
- Token de sesion en el dispositivo (almacenado de forma segura para mantener la sesion iniciada)

## 3. Finalidades del tratamiento

Usamos los datos para:

1. Crear y gestionar tu cuenta.
2. Permitir reservas de pistas y gestion de partidos.
3. Mostrar perfiles y funcionalidades sociales (favoritos/valoraciones).
4. Mantener la seguridad, prevenir fraude y resolver incidencias tecnicas.
5. Cumplir obligaciones legales aplicables.

## 4. Base legal (RGPD)

Tratamos datos personales sobre la base de:

1. **Ejecucion del contrato** (prestacion del servicio de la App).
2. **Interes legitimo** (seguridad, mejora operativa y prevencion de abuso).
3. **Cumplimiento de obligaciones legales**.
4. **Consentimiento**, cuando sea necesario para funcionalidades concretas.

## 5. Comparticion de datos con terceros

No vendemos tus datos personales.

Podemos compartir datos con proveedores necesarios para operar el servicio:

1. **Infraestructura y base de datos**
- Render (hosting del backend)
- Supabase/PostgreSQL (almacenamiento de datos)

2. **Integracion opcional de calendario**
- Google Calendar API (solo cuando la integracion esta activa para sincronizar eventos de reserva/partido)

Estos proveedores actuan como encargados del tratamiento segun sus propios terminos y medidas de seguridad.

## 6. Transferencias internacionales

Algunos proveedores pueden procesar datos fuera del Espacio Economico Europeo.  
Cuando aplique, se adoptan mecanismos legales adecuados (por ejemplo, clausulas contractuales tipo o equivalentes).

## 7. Conservacion de datos

Conservamos los datos mientras:

1. Tu cuenta este activa.
2. Sean necesarios para prestar el servicio.
3. Existan obligaciones legales que exijan su conservacion.

Cuando dejan de ser necesarios, se eliminan o anonimizan de forma segura.

## 8. Seguridad

Aplicamos medidas tecnicas y organizativas razonables para proteger la informacion, incluyendo:

1. Almacenamiento de contrasenas con hash.
2. Control de acceso y autenticacion mediante token.
3. Restriccion de acceso a base de datos en entorno servidor.
4. Monitorizacion basica de errores e incidencias.

Ningun sistema es 100% invulnerable; aun asi trabajamos para reducir riesgos de forma continua.

## 9. Derechos de las personas usuarias

Puedes ejercer, segun la normativa aplicable, tus derechos de:

1. Acceso.
2. Rectificacion.
3. Supresion.
4. Oposicion.
5. Limitacion del tratamiento.
6. Portabilidad.

Para ejercerlos, escribe a: `sergio.hlara84@gmail.com`

## 10. Menores de edad

La App no esta dirigida especificamente a menores de 14 anos.  
Si detectamos tratamiento no autorizado de datos de menores, adoptaremos medidas para su eliminacion.

Publicaremos la version vigente con su fecha de actualizacion.
